Labs

Silly little projects made in my spare time.
Photo: Flickr/Yesmoke

  • T9 Predictive Text Input Emulator. Words prediction from numerical keypad input.

  • pyWRFChemEmiss. WRF-Chem data emission preparation tool. Convert and interpolate emission data from excel speadsheets into NetCDF format suitable for WRF-Chem.

  • Thorn. A wind rose plot generator.

  • AERMOD Linux Port. AERMOD is US EPA's pollutant dispersion model. This version is tweaked to compile under gfortran.

  • Rotate. A simple backup files rotator.

  • log-analyzers. Extract visitor data from Apache & Nginx log files.
