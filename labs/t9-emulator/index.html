T9 Predictive Text Input Emulator

Open in New Window

This is a T9 Predictive Text Input Emulator, implemented using Trie.

Hint: use cycle button to cycle through all matching words.

Limitation

  • It’s not context-aware or grammar-aware. No smart result ordering.
